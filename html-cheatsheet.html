HTML Cheat Sheet

Basic Structure

        
        
          <!doctype html>
          <html>
          <head>...</head>
          <body>...</body>
          </html>

Common Tags

  • <h1> - <h6> headings
  • <p> paragraph
  • <a href=""> link
  • <img src="" alt=""> image
  • <div> generic container
  • <section> section of a page

Lists

        
          <ul>
          <li>Item</li>
          </ul>

          <ol>
          <li>First</li>
          </ol>

Forms

        
          <form>
          <label>Name</label>
          <input type="text" />

          <label>Message</label>
          <textarea></textarea>

          <button>Send</button>
          </form>

Tables

        
          <table>
          <tr>
          <th>Title</th>
          <th>Value</th>
          </tr>
          <tr>
          <td>Row 1</td>
          <td>Data</td>
          </tr>
          </table>
