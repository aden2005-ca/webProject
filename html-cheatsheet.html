HTML Cheat Sheet

Basic Structure

        
        
          <!doctype html>
          <html>
          <head>...</head>
          <body>...</body>
          </html>

Common Tags

  • <h1> - <h6> headings
  • <p> paragraph
  • <a href=""> link
  • <img src="" alt=""> image
  • <div> generic container
  • <section> section of a page

Lists

        
          <ul>
          <li>Item</li>
          </ul>

          <ol>
          <li>First</li>
          </ol>

Forms

        
          <form>
          <label>Name</label>
          <input type="text" />

          <label>Message</label>
          <textarea></textarea>

          <button>Send</button>
          </form>

Meta Viewport

Helps page scale correctly on mobile devices.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Semantic Tags

Makes page structure clearer and easier to understand.

<header><nav><main><footer>

Button Element

Used for actions like submit or click events.

<button>Click</button>

Input Types

Different input types help validation.

<input type="email">
    <input type="number">

Label Tag

Improves form accessibility.

<label for="name">Name</label>

Alt Attribute

Shows text if image fails and helps screen readers.

<img src="img.jpg" alt="sample image">

Anchor Links

Used for navigation between pages.

<a href="page.html">Go</a>

Figure Tag

Groups image and caption.

<figure>
      <img>
      <figcaption>text</figcaption>
    </figure>

Details Tag

Creates expandable content.

<details>
        <summary>More</summary>
      </details>

HTML Entities

Used to display reserved characters.

&lt; &gt; &amp;

Tables

        
          <table>
          <tr>
          <th>Title</th>
          <th>Value</th>
          </tr>
          <tr>
          <td>Row 1</td>
          <td>Data</td>
          </tr>
          </table>
