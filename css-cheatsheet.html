CSS Cheat Sheet

Selectors

        
          h1 { color: red; }
          .box { padding: 10px; }
          #main { width: 100%; }

Box Model

  • margin outside space
  • border line around
  • padding inside space
  • content the actual element content

Flexbox

        
          .row {
          display: flex;
          gap: 10px;
          justify-content: space-between;
          align-items: center;
          }

Grid

        
          .grid {
          display: grid;
          grid-template-columns: repeat(3, 1fr);
          gap: 12px;
          }

Responsive

        
          @media (max-width: 700px) {
          .nav { display: none; }
        }

Meta Viewport

Helps page scale correctly on mobile devices.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Semantic Tags

Makes page structure clearer and easier to understand.

<header><nav><main><footer>

Button Element

Used for actions like submit or click events.

<button>Click</button>

Input Types

Different input types help validation.

<input type="email">
<input type="number">

Label Tag

Improves form accessibility.

<label for="name">Name</label>

Alt Attribute

Shows text if image fails and helps screen readers.

<img src="img.jpg" alt="sample image">

Anchor Links

Used for navigation between pages.

<a href="page.html">Go</a>

Figure Tag

Groups image and caption.

<figure>
      <img>
      <figcaption>text</figcaption>
    </figure>

Details Tag

Creates expandable content.

<details>
      <summary>More</summary>
    </details>

HTML Entities

Used to display reserved characters.

&lt; &gt; &amp;
