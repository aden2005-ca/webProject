CSS Cheat Sheet

Selectors

        
          h1 { color: red; }
          .box { padding: 10px; }
          #main { width: 100%; }

Box Model

  • margin outside space
  • border line around
  • padding inside space
  • content the actual element content

Flexbox

        
          .row {
          display: flex;
          gap: 10px;
          justify-content: space-between;
          align-items: center;
          }

Grid

        
          .grid {
          display: grid;
          grid-template-columns: repeat(3, 1fr);
          gap: 12px;
          }

Responsive

        
          @media (max-width: 700px) {
          .nav { display: none; }
        }
