Variables

          const age = 20;

Functions

        
          function add(a, b) {
            return a + b;
          }

          const sub = (a, b) => a - b;

Arrays

        
          let nums = [1,2,3];
          nums.push(4);

          for (let i=0; i<nums.length; i++) {
            console.log(nums[i]);
          }

DOM

        
          const btn = document.getElementById("myBtn");
          btn.addEventListener("click", function(){
            alert("clicked");
          });

Data Types

JS has string, number, boolean.

let age = 20;

Equality

Strict equality checks value and type.

a === b

Template Strings

Allows variables inside strings.

`Hello ${name}`

Objects

Stores data in key-value pairs.

let user = {name:"Ali"};

Loops

Used to repeat actions.

for(let i=0;i<5;i++)

Map Function

Transforms array values.

arr.map(x => x*2)

Filter

Selects values from array.

arr.filter(x => x>5)

Query Selector

Selects DOM elements.

document.querySelector(".box")

ClassList

Adds or removes classes.

el.classList.add("active")

Try Catch

Handles runtime errors.

try { } catch(e) { }

Conditions

        
          if (age >= 18) {
          console.log("adult");
          } else {
          console.log("minor");
          }
